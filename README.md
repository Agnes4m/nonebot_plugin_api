## 使用说明

    目前只支持直接出现结果的图片或者文字

## 操作方法

    启动一次，在bot目录下data/api/api.json，填入url和其他信息格式如下

        {
                "data":[
                        {
                        "detail":"这里是注释",
                        "command":"涩图",
                        "main_url":"http://114514:514?key=114&value=514"
                        }
                ]
        }
